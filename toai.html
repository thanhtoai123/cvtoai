KỸ NĂNG

  • HTML CSS
    • Kiến thức về HTML:
      • Biết tư duy phân tích thiết kế khung layout.
      • Nắm được các bộ thẻ cốt lõi để xây dựng khung HTML
    • Kiến thức về CSS:
      • Các bộ chọn trong CSS như: selector id, selector css...
      • Các thuộc tính cho text, thuộc tính định dạng cho khối.
      • Biết bố cục trang với Flexbox.
      • Các plugin quan trọng như: thêm google font, fontawesome...
    • Biết chuyển sử dụng Photoshop chuyển PSD thành HTML
  • JQUERY
    • Nắm được các kiến thức cơ bản về Jquery như:
      • Sự kiện: Submit form, Click, Scroll, Resize,...
      • Hiệu ứng: Fade in/out, slideUp, slideDown, hide, show,...
      • Ajax: hiểu cách thức hoạt động của Ajax
  • BOOTSTRAP
    • Kiến thức Bootstrap:
      • Xây dựng layout website responsive với Grid System.
      • Nắm được các Class cơ bản để xây dụng giao diện nhanh hơn. Sử dụng được các cấu trúc để xây dụng khung trong HTML
  • PHP
    • Kỹ năng và kiến thức:
      • Nắm được dữ liệu trong php, những phép toán trong php, cấu trúc điều khiển và vòng lặp
      • Nắm được các kiểu dữ liệu truyền lên server: POST và GET.
      • Biết chuẩn hóa dữ liệu form.
      • Hiểu phiên làm việc: COOKIE, SESSION.
      • Hiểu được cách hoạt động của mô hình MVC
      • Hiểu cách làm việc giữa PHP với Ajax...
      • Biết làm việc với Hệ quản trị cơ sở dữ liệu Mysql
  • LARAVEL
    • Kiến thức:
      • Biết lập trình hướng đối tượng OOP.
      • Nắm được luồng xử lý MVC trong Laravel.
      • Tạo bảng với Migration.
      • Hiểu rõ và nắm được cách làm việc với Database Query Builder và Eloquent ORM.
      • Xử lý dữ liệu quan hệ trong Database với Eloquent ORM.
      • Quản lý Usermanager.
      • Kiểm soát truy cập người dùng qua MiddleWare.
